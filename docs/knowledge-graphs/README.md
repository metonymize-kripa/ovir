# Knowledge Graph History: Survey Index

A reading list and annotated survey of the knowledge graph field, organized chronologically by the literature each file covers.

| File | Covers | Status |
|------|--------|--------|
| [01-overview-peng2023.md](./01-overview-peng2023.md) | Peng et al. (2023) — systematic survey of KG opportunities and challenges; definitions, historical provenance (2004–2016), embedding taxonomy, open problems | Done |
| [02-history-chaudhri2022.md](./02-history-chaudhri2022.md) | Chaudhri et al. (2022) — dedicated history of KGs from AI Magazine; traces semantic networks (1956) → Cyc (1984) → Semantic Web → modern KGs; contrasting perspectives from 11 authors across Cyc, Wikidata, enterprise data integration, and linked data traditions | Done |
| [03-canonical-hogan2021.md](./03-canonical-hogan2021.md) | Hogan et al. (2021) — ACM Computing Surveys (18 authors, 4000+ citations); canonical technical reference; formal graph data model taxonomy (DEL/heterogeneous/property/dataset), query languages (SPARQL, Cypher, Gremlin), schema types, ontology reasoning (OWL profiles), embedding families, GNNs, rule mining, creation/quality/publication lifecycle, 30+ named open and enterprise KGs | Done |
| [04-embeddings-ji2021.md](./04-embeddings-ji2021.md) | Ji et al. (2021) — IEEE TNNLS survey (4000+ citations); four-view KGE taxonomy (representation space / scoring function / encoding model / auxiliary information); knowledge acquisition: NER, entity linking, entity alignment, relation extraction; temporal KGs; commonsense KGs; KGQA; KG-enhanced recommendations | Done |

**Reading order for a new reader:** start with `01` for the map of the field, then `03` (Hogan) for rigorous definitions, then `02` (Chaudhri) for historical depth, then `04` (Ji) for embedding detail.
